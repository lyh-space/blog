SM-1 | 热力学创世纪

发表于 2023-09-13 更新于 2023-09-14 分类于 Physics ， Statistical Physics

我坚信，就其内容的普适性而言，热力学定律是唯一最具普适性的物理学理论，在其基本概念的运用范围和运用架构之内，热力学定律永远不可能被颠覆。

——阿尔伯特·爱因斯坦

起初，神创造了热力学系统和平衡态

平衡态由状态变量描写，状态变量已经有几何变量、力学变量、电磁变量、化学变量四类，神又把它们分为广延量和强度量两类

神发现还不够

神说，要有温度，于是就有了热力学第零定律

神通过第零定律定义了温度

神又在温度和其它状态变量之间建立了物态方程，但是神没有说方程的具体形式

神为能量的传递创造了两种形式：功和热

神发现准静态过程的功容易算，非静态过程只有一些特殊情况可以表达出来

神为了看到物体的吸热能力，赋予其热容的属性

神说，能量要守恒，于是就有了热力学第一定律

神通过第一定律创造出了内能

神又创造出了理想气体为凡人演示

神说，热现象要有方向，于是就有了热力学第二定律

神用卡诺循环为凡人演示第二定律

众神用各自的语言表述第二定律

神通过第二定律创造出了熵

神说，不可到达绝对零度，于是就有了热力学第三定律

神算了经典熵，发现可以取到负无穷，于是神用量子统计解释第三定律

神为了方便，给第一定律做了Legendre变换，于是创造出了 \(H\)、\(F\) 和 \(G\) 三个热力学函数

 

 

[1.1] 通过第零定律定义温度

假设有 A、B、C 三个系统，其状态分别用 \(\{A_1, A_2, A_3, \cdots\}\)、\({B_1, B_2, B_3, \cdots}\)、\({C_1, C_2, C_3, \cdots}\) 描述。

系统 A 与 C 处于平衡态，即

\[ f_{AC} \left( A_1, A_2, A_3, \cdots; C_1, C_2, C_3, \cdots \right) =0 \]

B 与 C 处于平衡态

\[ f_{BC} \left( B_1, B_2, B_3, \cdots; C_1, C_2, C_3, \cdots \right) =0 \]

因此有

\[ f_{AC} \left( A_1, A_2, A_3, \cdots; C_1, C_2, C_3, \cdots \right) = f_{BC} \left( B_1, B_2, B_3, \cdots; C_1, C_2, C_3, \cdots \right) \]

根据热力学第零定律，此时 A 与 B 也必然处于平衡态，即

\[ f_{AB} \left( A_1, A_2, A_3, \cdots; B_1, B_2, B_3, \cdots \right) =0 \]

可得

\[ \Theta_{A}\left( A_1, A_2, A_3, \cdots \right) = \Theta_B \left( B_1, B_2, B_3, \cdots \right) \]
